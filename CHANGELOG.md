# Changelog

## 1.2.0

- Apple Pencil Support - Write numbers directly into cells using Apple's Scribble feature. A natural, handwriting-first experience on iPad.
- Haptic Feedback - Feel subtle vibrations on button presses and when placing correct (or incorrect!) numbers.
- Improved Localization - Better translations for non-English speakers around the world.
- Various UI fixes and polish
- Dependency updates and general cleanup
